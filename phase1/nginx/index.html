Cadanac is a SAAS Virus Case Management System including Financial and Location Tracking Capabilities. It is developed in response to Covid19 and can accommodate dual and multiple infections (for example the Flu and Covid19 on the same host).

Read and follow the updates on LinkedIn: https://www.linkedin.com/pulse/cadanac-system-method-tracking-covid19-multiple-host-hrad-hekimian. All instructions on how to build and deploy Cadanac are there.

Phase 1 of the code to build, deploy and run Cadanac is 100% Open Source and lives here: https://github.com/hrad-h/cadanac/tree/master/phase1
